https://react-product-listing-omega.vercel.app/

## GitHub Repository

https://github.com/pushpender1330/react-product-listing

## Run Locally

Clone the repository:

```bash
git clone https://github.com/pushpender1330/react-product-listing.git
```

Navigate to the project folder:

```bash
cd react-product-listing
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open the application in your browser:

```bash
http://localhost:5173
```
